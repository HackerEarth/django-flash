**************************************
Welcome to Django Flash documentation!
**************************************

Djnago flash is a cache framework. It features:

* Easy to create cache classes for mostly used cache queries
* Totally consistent
* Automatic invalidation
* Free from race conditions
* Option to make batch get queries
* Glues seamlessly with Python3 asyncio & dataloaders
* Dynamic versioning
